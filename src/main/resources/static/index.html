OpenShift

Run Docker container workloads with ease using OpenShift 3.7. Built on Kubernetes, OpenShift removes the frustration and headache of container orchestration and management and lets you focus on writing code and getting more done.

ZFS Storage

All data protected by fully redundant ZFS storage servers which continually monitor your data for corruption. Work load is spread across many disks to improve performance, and multiple copies are stored in redundant chassises for reliability.

Powerful Infrastructure

Multiple Data Centers, Redundant Firewalls, 10GB Networking, SSD backed storage, and more than gratuitous amounts of memory keep your applications running at top speeds. Backup generators on site provide power for extended outages.

Diverse Connections
Redundant Network

Networking into ShackSpace Hosting facilities is provided by two diverse connections. One copper based and the other fiber based, from different providers with different trenches to ensure that if one connection is cut, another will remain active in its place.
